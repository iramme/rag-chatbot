# ◈ DocMind — RAG Document Chatbot

An intelligent document chatbot built with RAG (Retrieval-Augmented Generation),
LangChain, LLaMA 3.3 and ChromaDB. Ask questions about your PDF documents
and get accurate answers with source citations.

🔗 **Live Demo :** [HuggingFace Spaces](https://huggingface.co/spaces/VOTRE_USERNAME/docmind-rag-chatbot)

---

## Features

- Upload multiple PDF documents
- Automatic text chunking and vectorization
- Semantic search with ChromaDB
- Answers generated by LLaMA 3.3 (via Groq API)
- Source citations with page numbers
- Dark / Light mode toggle
- Clean and minimal UI

## Tech Stack

| Component | Technology |
|-----------|-----------|
| Framework | LangChain 1.3 |
| LLM | LLaMA 3.3 70B (Groq) |
| Embeddings | all-MiniLM-L6-v2 (HuggingFace) |
| Vector Store | ChromaDB |
| PDF Loader | PyPDFLoader |
| UI | Streamlit |
| Deployment | HuggingFace Spaces |

## Installation

```bash
# Clone the repository
git clone https://github.com/VOTRE_USERNAME/docmind-rag-chatbot
cd docmind-rag-chatbot

# Create virtual environment
python -m venv venv
venv\Scripts\activate  # Windows
source venv/bin/activate  # Mac/Linux

# Install dependencies
pip install -r requirements.txt

# Create .env file
echo GROQ_API_KEY=your_key_here > .env

# Run the app
streamlit run app.py
```

---

## Usage

1. Upload your PDF files in the sidebar
2. Click **Build KB** to index the documents
3. Ask any question in the chat input
4. Get answers with source page references

## Get a Free Groq API Key

1. Go to [console.groq.com](https://console.groq.com)
2. Create a free account
3. Click API Keys → Create API Key
4. Add it to your `.env` file
